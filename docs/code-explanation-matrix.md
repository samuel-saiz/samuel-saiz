# 🟢 Matrix Landing Page - Samuel

This project is a "Matrix-style" landing page built using core web technologies, focused on creating a dynamic real-time animation effect.

## 🚀 Technologies Used

### 🧱 HTML5
Used to structure the web page.
- Document definition with `<!DOCTYPE html>`
- Semantic tags such as `<head>`, `<body>`, `<canvas>`, and `<div>`
- Central presentation message

### 🎨 CSS3
Used to design the visual Matrix-style aesthetic.
- Black background
- `monospace` typography
- Neon green color (`#00ff00`)
- Glow effect using `text-shadow`
- Absolute positioning to center content
- Responsive layout

### ⚡ JavaScript (Vanilla JS)
Used to generate the dynamic animation inside the `<canvas>` element.

Main features:
- Canvas API manipulation
- Random binary number generation (0 and 1)
- Animation using `setInterval`
- Trail effect using semi-transparent background
- Automatic resizing adaptation

## 🎯 Project Features

- Smooth digital rain animation
- Real-time random binary generation
- Matrix-inspired visual effect
- Minimalist and visually focused design
- Built without external libraries

## 📚 Concepts Applied

- DOM Manipulation
- Canvas API
- JavaScript Events
- Responsive Design
- 2D Animation
